#+TITLE: NixOS Configuration
#+AUTHOR: aRenCoco

* 简介
这是一个基于 Flake 的 NixOS 配置仓库，主机配置名为 =nixos=.

我自己打包的 nixos packages 仓库见 [[https://github.com/ren-lingyu/nixpkgs]]. 

* 目录结构

├── flake.lock

├── hosts

* 关键点
- [[file:hosts][hosts]] 放置设备相关的系统级配置.
- [[file:users][users]] 放置用户级配置.
- [[file:modules][modules]] 放置便于模块化的系统级配置. 
- =nixpkgs= 使用 =nixos-unstable= (清华镜像). 
- 启用 =nix-command= 与 =flakes=. 
- Home Manager 管理 zsh/git/ssh/emacs/aichat 等开发环境. 

* 使用
应用系统配置：
#+begin_src bash
sudo nixos-rebuild switch --flake .#nixos
#+end_src

更新依赖锁：
#+begin_src bash
nix flake update
#+end_src

* 脚本
- =org=: 快捷调用有关脚本, 同步 =~/org= 到 nutstore 等.
- =ssha=: 启动 =ssh-agent= 加载默认私钥.
