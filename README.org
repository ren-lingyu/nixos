* 简介
这是一个基于 Flake 的 NixOS 配置仓库，主机配置名为 =nixos=.

我自己打包的 nixos packages 仓库见 [[https://github.com/ren-lingyu/nixpkgs]].

* 关键点
- [[file:modules][modules]] 放置 NixOS 与 home-manager 模块, 按 =**/os/= (系统级), =**/hm/= (用户级), =**/lib/= (模块内部函数) 组织.
- [[file:modules/hosts][modules/hosts]] 放置设备相关配置, 含 thinkbook (bare metal NixOS, hostname =nixos=) 和 wsl (NixOS-WSL, hostname =nixos-wsl=).
- [[file:modules/users][modules/users]] 放置用户 home-manager 配置, 包含 nixvim/emacs/zsh/git/ssh/gpg/keepassxc/rclone/thunderbird/direnv 等.
- =nixpkgs= 使用 =nixos-unstable=, 另有自行打包的 [[https://github.com/ren-lingyu/nixpkgs][ren-lingyu/nixpkgs]] overlay.
- 启用 =nix-command= 与 =flakes=.
- 使用 =sops-nix= 管理密钥, 系统级秘密以 host ssh key 加密, 用户级秘密以独立的 age key 加密.
- 使用 =niri= 作为窗口管理器, 另有 =desktop-environment= 提供 gnome/hyprland 备选.

* 使用
重建系统:
#+begin_src bash
sudo nixos-rebuild switch --flake .#nixos
#+end_src

更新依赖:
#+begin_src bash
nix flake update
#+end_src

* 脚本
- =org=: 进入 =~/org= 目录并准备 make 函数, 支持 --texlive 调用 TeX Live 工具.
- =rc=: rclone 同步包装器, 支持 push/pull 子命令, 限制传输并发并显示进度.
- =ssha=: 启动 =ssh-agent= 并加载默认 ed25519 私钥.
