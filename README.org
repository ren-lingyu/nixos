* 关键点
- [[file:modules/features][modules/features]] 按功能领域组织 NixOS 与 home-manager 模块, 每个 feature 含 =default.nix= 作为入口, 并按 =os/= (系统级), =hm/= (用户级), =lib/= (模块内部函数) 分层.
- [[file:modules/hosts][modules/hosts]] 放置设备相关配置, 含 thinkbook (bare metal NixOS, hostname =nixos=), wsl (NixOS-WSL, hostname =nixos-wsl=) 和 aliyun (阿里云 ECS, hostname =nixos-aliyun=).
- [[file:modules/users][modules/users]] 放置用户 home-manager 配置, 包含 nixvim/emacs/zsh/git/ssh/gpg/keepassxc/rclone/thunderbird/direnv 等.
- [[file:modules/overlays][modules/overlays]] 放置自定义 nixpkgs overlay, 覆盖上游 =qq= 和 =wechat= 等 unfree 软件的版本与下载链接, 修复因腾讯官方链接变更导致的构建失败.
- =nixpkgs= 使用 =nixos-unstable=, 另有自行打包的 [[https://github.com/ren-lingyu/nixpkgs][ren-lingyu/nixpkgs]].
- 启用 =nix-command= 与 =flakes=.
- 使用 =sops-nix= 管理密钥, 系统级秘密以 host ssh key 加密, 用户级秘密以独立的 age key 加密.
- 使用 =niri= 作为窗口管理器.

* 脚本
- =org=: 进入 =~/org= 目录并准备 make 函数, 支持 --texlive 调用 TeX Live 工具.
- =rc=: rclone 同步包装器, 支持 push/pull 子命令, 限制传输并发并显示进度.
- =ssha=: 启动 =ssh-agent= 并加载默认 ed25519 私钥.
